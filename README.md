# Vorm - Advanced Vue Forms

## Simple Fields

* Boolean
  * label
  * value true false
  * default false

* String
  * label
  * value some string
  * default ""

* Text
  * String
  * label
  * value some string
  * default ""
  
* Number
* File
* Color (HTML 5 Color)

## Advanced Fields

* Markdown
* Code
* Wysiwyg (tiptap integration)
* Where When
* Location Maps (Google)

## Pickers

* DateTime     (Time and Date Picker)
* File         (sinle or multiple)
* Choise       (single or multiple)
* Select       (single or multiple)
* ButtonGroup  (single or multiple)
* Search       (autocomplete)
* MemorySearch (autocomplete with tagcloud)

## Advanced Groups

* Relation
* Object
* Blocks
* List
